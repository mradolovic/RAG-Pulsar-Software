The core pulsar detection and analysis program. It reads the channelised binary file produced by RTLChannel4bin, folds the data at a trial period, and searches over dispersion measure, period, and period derivative (p-dot) to find the best-fit pulsar candidate. For each trial it computes a signal-to-noise ratio using a Gaussian matched filter shaped to the expected pulse width. Results are written to a large set of text files — folded profiles, DM search curves, period/p-dot search grids, band waterfalls, section SNR timelines, and a 2D period/p-dot map — which are then read and plotted by pul_plot.py.
