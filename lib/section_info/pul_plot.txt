Reads all the text output files produced by pulsar_det_an and renders them as a set of diagnostic plots. It produces a folded pulse profile, DM search curve, period and p-dot search plots, band SNR chart, cumulative SNR timeline, rolling average SNR, compressed data waterfall, and a 2D period/p-dot image. All plots are saved as PNG files and stitched together into a single combined image for easy review. It can be run interactively or headlessly via the --no-show flag, in which case it uses a non-display matplotlib backend.
