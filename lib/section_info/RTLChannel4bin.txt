Takes a raw 8-bit IQ binary file from an RTL-SDR receiver and converts it into a channelised 4-byte float binary file ready for pulsar folding. It reads successive blocks of IQ samples, applies an N-point FFT to split the band into frequency channels, averages a configurable number of FFT frames in time to downsample the data rate, then writes each averaged spectrum as a row of 4-byte floats to the output file. The number of channels, the input sample rate, and the output data rate are all set on the command line.
